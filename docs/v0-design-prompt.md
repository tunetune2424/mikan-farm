# 伊佐みかん園 - v0.dev デザインプロンプト

v0.dev ( https://v0.dev ) のチャットにそのまま貼り付けて使用する。

---

```
Create a warm, handcrafted-feel Japanese mikan (mandarin orange) farm website 
homepage for "伊佐みかん園" (Isa Mikan-en).

## Design Concept
- Warm, rustic, family-owned farm atmosphere
- Hand-grown with care in mountain countryside
- Natural textures (paper, wood grain hints)
- Color palette: warm orange (#E8732A), deep green (#3D6B35), 
  cream/off-white (#FDF6E3), brown accents (#7B4F2E)
- Fonts: rounded/friendly sans-serif for headings, 
  clean readable font for body

## Page Sections (single page scroll)

1. **Header/Nav**
   - Logo: "伊佐みかん園" with small mikan icon
   - Nav links: ホーム / 農家紹介 / 商品 / みかん狩り / こだわり / お問い合わせ
   - Sticky, semi-transparent on scroll

2. **Hero Section**
   - Full-width background: mikan orchard on mountain hillside 
     (use placeholder image)
   - Overlay with warm orange tint (opacity 30%)
   - Headline: "山の恵み、手でひとつひとつ。"
   - Sub: "愛媛の山奥で、家族みんなで育てたみかんをお届けします。"
   - CTA button: "商品を見る" (warm orange, rounded)
   - Second visual: close-up mikan photo floating/overlapping

3. **農家紹介 Section**
   - Split layout: photo left, text right
   - Warm heading: "伊佐みかん園について"
   - Description about family-run farm in mountains

4. **商品 Section**
   - 3 product cards in grid
   - Card style: rounded corners, subtle shadow, cream background
   - Each: image, name, description, price, "カートに追加" button

5. **みかん狩り Section**
   - Seasonal experience highlight section
   - Badge or ribbon: "期間限定"
   - Info cards layout:
     🍊 開催シーズン: 10月〜12月（みかんの収穫時期）
     💰 料金: 大人 1,500円 / 子ども 800円（食べ放題・1時間）
     📍 アクセス: 〇〇県〇〇市（山の中・駐車場あり）
   - CTA button: "予約・お問い合わせはこちら" linking to contact section
   - Warm seasonal background (orange gradient or autumn-feel)

6. **こだわり Section**
   - 3 feature icons with descriptions
   - 🍊 手作業収穫 / 🌿 農薬を減らした栽培 / 📦 産地直送

7. **お問い合わせ Section**
   - Simple contact form
   - Fields: お名前, メールアドレス, お問い合わせ内容
   - Note: みかん狩り予約もこちらから

8. **Footer**
   - Farm name, copyright, warm brown background

## Technical Requirements
- React functional components with Tailwind CSS
- Fully responsive (mobile-first)
- Use placeholder images from picsum.photos
- Smooth scroll between sections
- Hover effects on buttons and cards
```

---

## 使い方
1. v0.dev を開く
2. 上記コードブロックの中身をコピーしてチャットに貼り付ける
3. 気に入ったら「Export Code」でReactコードをコピー
4. `mikan-farm/src/` に組み込む

## 修正時の追加指示例
- 「もっと和風な雰囲気に」
- 「みかん狩りセクションをもっと目立たせて」
- 「料金を大人2,000円、子ども1,000円に変更して」
